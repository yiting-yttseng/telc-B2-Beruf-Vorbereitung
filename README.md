**Wortschatz-Tracker zur Prüfungsvorbereitung auf den Deutsch-Test für den Beruf B2 (DTB B2)**

Dieser Tracker wurde im Rahmen eines Berufssprachkurses der Bundesagentur für Arbeit erstellt.

**Verwendete Lehrwerke:**
- Klett Aspekte Beruf B1/B2 Brückenelement (ISBN 978-3-12-605362-4)
- Klett Aspekte Beruf B2  (ISBN 978-3-12-605361-7)

**Inhalt:**
- Wortschatz (alle Kapitel)
- Nomen-Verben-Verbindungen
- Verben / Nomen / Adjektive mit Präpositionen
- Prüfungsrelevante Ausdrücke und Redemittel

**Funktionen:**
- Vorlese-Funktion (Text-to-Speech) zum Shadow-Training
- Suchfunktion zur schnellen Abfrage
- Berufliche Beispielsätze
