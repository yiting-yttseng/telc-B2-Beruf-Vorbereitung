# telc-B2-Beruf-Vorbereitung
Vocabulary tracker for telc DTB B2 exam prep — covering Wortschatz, Nomen-Verben-Verbindungen, Verben/Nomen/Adjektive mit Präpositionen, and exam-style phrases from Klett Aspekte Beruf B1/B2 &amp; B2. Features text-to-speech, search, and example sentences. Course funded by Bundesagentur für Arbeit.
